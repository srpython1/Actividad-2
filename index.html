Estructuración de una Página Web

Tablas en HTML

Concepto Descripción
Etiqueta table Define una tabla en HTML.
Atributos de table Atributos como border, cellspacing, y cellpadding para definir el estilo de la tabla.
Etiqueta tr Define una fila en la tabla.
Atributo tr Atributos como align y valign para alinear el contenido.
Etiqueta td y th Define celdas de datos (td) y celdas de encabezado (th).
Atributos de td y th Atributos como colspan y rowspan para combinar celdas.
Etiqueta thead Define el encabezado de la tabla.
Etiqueta tfoot Define el pie de la tabla.
Etiqueta tbody Define el cuerpo de la tabla.

Formulario en HTML

Concepto Descripción
Etiqueta form Define un formulario para recoger datos del usuario.
Etiqueta input Define un campo de entrada de datos.
Etiqueta label Asocia una etiqueta con un elemento de formulario.
Etiqueta select Define un menú desplegable.
Etiqueta textarea Define un área de texto para entrada de múltiples líneas.
Etiqueta fieldset Agrupa elementos de un formulario.
Etiqueta legend Proporciona un título para un fieldset.
Etiqueta meter Representa un valor en un rango conocido.
Etiqueta progress Indica el progreso de una tarea.
Etiqueta datalist Define una lista de opciones predefinidas para un campo de entrada.
Etiqueta summary Proporciona un resumen visible para un elemento details.

Estructuración de una Página Web

  • Contenedor (wrapper): Un elemento que envuelve todo el contenido de la página para aplicar estilos comunes.
  • Cabecera (header): Contiene el título de la página y elementos de navegación.
  • Contenido (content): La parte principal de la página donde se muestra la información.
  • Menú (menu): Una lista de enlaces que permiten navegar por la página o el sitio web.
  • Pie (footer): Sección al final de la página que suele contener información de contacto y derechos de autor.
  • Lateral (sidebar): Área adicional a un lado del contenido principal, que puede contener enlaces, información o anuncios.
  • Etiqueta div: Un contenedor genérico utilizado para agrupar otros elementos y aplicar estilos.

Estructuración Semántica de una Página Web

  • <header>: Contiene información introductoria y navegación.
  • <nav>: Define un conjunto de enlaces de navegación.
  • <main>: Representa el contenido principal de la página.
  • <article>: Define contenido independiente que puede ser distribuido de manera independiente (ej. un artículo de blog).
  • <section>: Agrupa contenido relacionado dentro del documento.
  • <aside>: Contenido relacionado, como una barra lateral o un bloque de información adicional.
  • <footer>: Contiene información de pie de página, como derechos de autor.
